# EPIC: MEDIA-005 Remote Control Layer

## Goal

Allow TV remote to trigger player commands.

## Input Source

- HDMI-CEC events

## Example Mappings

- PLAY/PAUSE -> pause
- NEXT -> skip
- UP -> like
- RIGHT -> add playlist
- LEFT -> queue
- BACK -> close UI
